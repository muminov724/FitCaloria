# Telegram Food Analyzer Bot 🍏🤖

Telegram-бот на Python, который анализирует фотографии еды с помощью искусственного интеллекта Google Gemini (модель `gemini-2.5-flash`), определяет блюдо или продукты и рассчитывает примерную калорийность, количество белков, жиров и углеводов (БЖУ).

## Возможности
- Анализ фотографий еды с использованием нейросетей Google Gemini.
- Расчет примерной пищевой ценности (КБЖУ: Калории, Белки, Жиры, Углеводы).
- Полностью асинхронный код с использованием `python-telegram-bot` и асинхронных библиотек.
- Защищенное хранение API ключей в файле `.env`.
- Обработка ошибок (некорректный формат разметки, недоступность API, отсутствие еды на фото).

## Требования
- Python 3.10 или выше.
- Токен Telegram-бота (от [@BotFather](https://t.me/BotFather)).
- API Ключ Gemini (от [Google AI Studio](https://aistudio.google.com/)).

## Установка и запуск

1. **Клонируйте репозиторий или перейдите в папку с проектом:**
   ```bash
   cd c:\Users\Hp\OneDrive\Desktop\manman
   ```

2. **Создайте и активируйте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для macOS/Linux:
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения:**
   Переименуйте файл `.env.example` в `.env` (или заполните созданный файл `.env`):
   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_телеграм_бота
   GEMINI_API_KEY=ваш_api_ключ_gemini
   ```

5. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Использование
1. Запустите бота в Telegram, введя команду `/start`.
2. Отправьте боту фотографию блюда (например, салат, пицца, суп).
3. Бот ответит вам сообщением с описанием блюда, его примерным весом, таблицей или списком БЖУ и калорий, а также парой полезных советов от виртуального диетолога.
